Program Name
Headlong SMS Notifications

Program Description
Headlong is an OSS project that allows users to set up SMS messaging via Twilio to enable communication between users and an AI agent. Messages are sent and received in response to user-initiated interactions.

Message Frequency
Message frequency varies based on user interaction and the version of Headlong they are running. Users only receive messages when they set up SMS communications with the agents they create.

Message and Data Rates
Message and data rates may apply. Please contact your wireless carrier for details about your messaging plan.

Support
For support, open an issue at: https://github.com/andyk/headlong/issues

Opt-Out
To stop receiving messages, stop your agent or delete your Twilio number.
